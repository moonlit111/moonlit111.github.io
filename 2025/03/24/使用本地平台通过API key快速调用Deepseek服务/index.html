使用开源软件Chatbox和Cherry Studio通过API快速调用Deepseek服务

Finland
2025-03-24
大语言模型, Deepseek

一. 选择API key提供商，并购买API key额度

以下提供商依照个人喜好排序

1. 火山引擎

  • 优点：极其低价（咸鱼0.5CNY->30CNY代金券）、较快的响应速度
  • 缺点：Chatbox对其支持较差、容易犯傻（貌似不是严格意义上的满血）

2. 硅基流动

  • 优点：响应速度快、与官网一致的满血体验
  • 缺点：价格较贵（新用户送14CNY代金券）

3. 深度求索

  • 优点：与官网一致的满血体验、官方提供的API key
  • 缺点：响应速度慢、没有送代金券、价格较贵

4. 百度云千帆

  • 优点：响应速度快
  • 缺点：价格跟官方一致，但模型疑似非满血、价格较贵

建议：

  1. 根据个人喜好，选择2-3个API key提供商
  2. 一部分提供商未体验过（如阿里云百练、腾讯混元），此处不予置评按需选择
